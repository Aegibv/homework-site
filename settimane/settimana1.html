Homework - Settimana 1

Fare il primo programma JavaScript che scrive un messaggio colorato, rimpiazzando il testo di un DIV già presente nella pagina. Il programma deve essere inserito in un file separato usando JavaScript esterno. Inoltre dimostrare l'algoritmo di Welford.

Risultato

Questo testo verrà sostituito da JavaScript.

Codice HTML


<div id="messaggio">
  Questo testo verrà sostituito da JavaScript.
</div>

<script src="../script.js"></script>

Codice JavaScript


// script.js

const div = document.getElementById("messaggio");

if (div) {
  div.textContent = "Ciao! Questo messaggio è scritto con JavaScript";

  div.style.fontWeight = "bold";
  div.style.fontSize = "24px";

  div.style.background = "linear-gradient(90deg, red, orange, yellow, green, blue, indigo, violet)";
  div.style.webkitBackgroundClip = "text";
  div.style.webkitTextFillColor = "transparent";
  div.style.color = "transparent";
}
