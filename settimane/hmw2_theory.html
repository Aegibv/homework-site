📚 Teoria: Generare distribuzioni partendo dall’Uniforme

I computer generano numeri pseudo-casuali con distribuzione Uniforme tra 0 e 1. Per ottenere altre distribuzioni, si applicano trasformazioni matematiche specifiche ai valori generati. In questo modo è possibile simulare distribuzioni continue e discrete a partire da una sorgente uniforme di base.

🟢Distribuzioni Continue

Esponenziale (Trasformata inversa)
Formula: X = - ln(U) / λ

Normale / Gaussiana (Metodo di Box-Muller)
Formula: Z₁ = √(-2 ln(U₁)) · cos(2πU₂) Z₂ = √(-2 ln(U₁)) · sin(2πU₂)

Cauchy (Trasformata inversa)
Formula: X = x₀ + γ tan(π(U - 0.5))

Chi-Quadro (χ²) (Composizione)
Formula: χ²k = Σ Zi2,   con Zi ~ N(0,1)

F di Fisher (Rapporto)
Formula: F = (χ₁² / ν₁) / (χ₂² / ν₂)

Triangolare (Trasformata inversa a tratti)
Formula: X = a + √(U(b-a)(c-a))   se   U < (c-a)/(b-a) X = b - √((1-U)(b-a)(b-c))   altrimenti

Pareto (Trasformata inversa)
Formula: X = xm(1 - U)-1/α

🔵Distribuzioni Discrete

Bernoulli (Soglia diretta)
Formula: X = 1 se U < p, altrimenti X = 0

Binomiale (Somma di Bernoulli)
Formula: X = X₁ + X₂ + ... + Xn dove ogni Xi ~ Bernoulli(p)

Geometrica (Trasformata inversa discretizzata)
Formula: X = ⌈ ln(U) / ln(1-p) ⌉

Poisson (Moltiplicazione sequenziale)
Algoritmo: L = e-λ Si moltiplicano U₁, U₂, ... finché il prodotto P < L
